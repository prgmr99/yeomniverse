# @hyo/ui

Shared UI components package for the Hyo Domain monorepo.

## Components

### Layout Components
- `Footer` - Application footer with policy links and copyright
- `Loading` - Loading spinner animation

### Analytics & Scripts
- `GoogleAdSense` - Google AdSense integration component
- `GoogleAnalytics` - Google Analytics tracking component
- `KakaoScript` - Kakao SDK script loader

## Usage

```typescript
import { Footer, Loading, GoogleAnalytics } from '@hyo/ui';
```

pnpm lint    # Run Biome linting
