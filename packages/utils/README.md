# @hyo/utils

Shared utility functions and constants for the Hyo Domain monorepo.

## Utilities

- `calculateResult` - Quiz result calculation logic
- `constants` - Shared constants and configuration values

## Usage

```typescript
import { calculateResult, QUIZ_QUESTIONS } from '@hyo/utils';
```

## Development

```bash
pnpm lint    # Run Biome linting
pnpm format  # Format code with Biome
```
